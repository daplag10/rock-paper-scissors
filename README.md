# 🎮 Rock Paper Scissors (Console Version)

A simple Rock–Paper–Scissors game built using **HTML and JavaScript**, running in the browser console with no graphical interface.

---

## 🚀 Features

* Play directly in the browser using `prompt()`
* Random computer opponent
* Score tracking (Player vs Computer)
* Continuous gameplay loop
* Exit anytime by typing `exit`

---

## 📂 Project Structure

```
index.html
```

---

## ▶️ How to Run

1. Open `index.html` in your browser
2. A prompt will appear asking for your move
3. Enter:

   * `rock`
   * `paper`
   * `scissors`
4. Open the browser console (**press F12 or Ctrl + Shift + J (Windows/Linux) / Cmd + Option + J (Mac)**) to see results
5. Type `exit` to end the game

---

## 🧠 How It Works

* The player enters a choice via prompt
* The computer generates a random choice
* The winner is determined using simple game logic
* Scores are updated and displayed in the console

---
